PivotLayoutManager

class PivotLayoutManager(properties: RecyclerView.LayoutManager.Properties) : RecyclerView.LayoutManager, ItemTouchHelper.ViewDropHandler, RecyclerView.SmoothScroller.ScrollVectorProvider

A RecyclerView.LayoutManager that builds the layout around a pivot view.

It behaves similarly to GridLayoutManager with the main difference being how focus is handled.
